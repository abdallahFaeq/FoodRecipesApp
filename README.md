Food Recipes App:
  It uses the API Service to display a group of different foods around the world, the method of preparation, and also save them offline.

Technologies:
  Kotlin language,
  MVC pattern,
  REST API (Retrofit),
  Room Database,
  Kotlin coroutines,
  Easy permission library.
